✅ 1. Importing and Reading a PDF
python
Copy
Edit
from pypdf import PdfReader

reader = PdfReader("example.pdf")





📃 3. Extract Text from a Page
python
Copy
Edit
page = reader.pages[0]         # get first page (index starts from 0)
text = page.extract_text()
print(text)



🔍 4. Read Metadata
python
Copy
Edit
info = reader.metadata
print(info.title)              # Title of the document
print(info.author)             # Author of the document
print(info.producer)           # Who produced the PDF




🔐 5. Check for Encryption
python
Copy
Edit
print(reader.is_encrypted)
If True, the PDF is password protected.



🔓 6. Decrypt a Password-Protected PDF //imp 
python
Copy
Edit
if reader.is_encrypted:
    reader.decrypt("your_password")
🔴 Decryption only works if you know the password.





📌 7. Get Page Size and Layout Info
python
Copy
Edit
page = reader.pages[0]
print(page.mediabox)          # Page dimensions (MediaBox)
print(page.cropbox)           # CropBox (visible part)
print(page.rotate)            # Rotation angle




📂 8. Extract Images (Advanced)
PyPDF does not extract images directly, but you can access raw XObjects:

python
Copy
Edit
xObject = page["/Resources"]["/XObject"].get_object()

for obj in xObject:
    if xObject[obj]["/Subtype"] == "/Image":
        data = xObject[obj].get_data()
        # Save or process the image
For image extraction, better use pdfminer.six or fitz (PyMuPDF).




📑 9. Get Outline / Table of Contents
python
Copy
Edit
outline = reader.outline
